# PY_Calculator

Simple Scientific Calculator made with Python3.6

## Getting Started

Just Run main.py

### Prerequisites
Python 3.6
Python Module : tkinter, pylab, PIL, numpy, sympy (you download use Anaconda)
For install the modulema manually, you can write in your System Shell:

```
pip install tkinter
pip install pylab
pip install PIL
pip install PIL
pip install numpy
pip install Sympy
```
